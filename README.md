# settle
A home-grown browser-based game insired by early The Settlers games.

## Random thoughts
How to keep combat to a minimum but still make it engaging and challenging?

- Co-op game against the AI
- Resource gathering - race for resources agains AI
- Maybe simply block oponent from resources without full-on war?
- Trade?

## Fun things to have
- Many animated/mobile units on screen at once
- Farms, growing resources
- Tree planting and logging
- Mining materials
- Building structures that refine materials

## Tech stack
- html, vanilla js and css
- php server to manage loading and saving
- sockets to manage interactions between users and server
- spacial hash grid for collisions
- service worker to seperate back-end processing with front-end rendering
